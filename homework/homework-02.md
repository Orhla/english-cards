# Домашнее задание 2: Аудио и голос

## Контекст

В прошлый раз мы сделали карточки с переключением и переворотом. Сейчас добавим два новых взаимодействия:

- **Послушать** как слово произносится — кнопка динамика на лицевой стороне
- **Произнести перевод по-русски** — вместо того чтобы просто переворачивать карточку кликом, нажимаем микрофон, говорим перевод, карточка переворачивается и показывает попали мы или нет

---

## Часть 1: Яндекс SpeechKit

Зайди в [Яндекс Облако](https://cloud.yandex.ru), создай аккаунт и получи **API-ключ** для SpeechKit.

Добавь в `.env`:
```
YANDEX_API_KEY=your_key_here
```

Убедись что `.env` в `.gitignore` — ключ в репозиторий попасть не должен.

---

## Часть 2: TTS-скрипт

Создай файл `scripts/generate-audio.ts`.

Скрипт должен:
1. Достать все карточки из базы через Prisma
2. Для каждой карточки проверить — существует ли уже файл `public/audio/{word}.ogg`
3. Если файл есть — пропустить (идемпотентность)
4. Если нет — сходить на Яндекс TTS API и сохранить файл

**Яндекс TTS API:**
```
POST https://tts.api.cloud.yandex.net/speech/v1/tts:synthesize
Authorization: Api-Key {YANDEX_API_KEY}
Content-Type: application/x-www-form-urlencoded

text=hello&lang=en-US&format=oggopus&voice=john
```

Ответ — бинарные данные аудио, пишешь их напрямую в файл.

Добавь скрипт в `package.json`:
```json
"generate-audio": "ts-node scripts/generate-audio.ts"
```

Запусти `npm run generate-audio` — в папке `public/audio/` должны появиться `.ogg` файлы.

> Загляни в документацию SpeechKit — там есть список голосов и языков. Выбери тот что нравится.

---

## Часть 3: Кнопка динамика

На лицевой стороне карточки добавь кнопку 🔊.

По клику — воспроизводит `/audio/{word}.ogg`. В браузере это делается через `new Audio(src).play()`.

**Важный момент:** клик на кнопку не должен переворачивать карточку. Подумай как это решить.

---

## Часть 4: Голосовой ввод

На лицевой стороне карточки добавь кнопку микрофона 🎤.

По клику запускается распознавание речи. Студентка говорит перевод **по-русски** — карточка автоматически переворачивается.

Для распознавания используй **Web Speech API** — это браузерное API, библиотеки не нужны:

```ts
const recognition = new window.webkitSpeechRecognition()
recognition.lang = 'ru-RU'
recognition.onresult = (event) => {
  const transcript = event.results[0][0].transcript
  // дальше твоя логика
}
recognition.start()
```

> Web Speech API работает только в Chrome — для домашки это нормально.

На обратной стороне карточки покажи:
- Что студентка сказала
- Правильный перевод
- Попала или нет ✅ / ❌

Сравнение строк будет неточным — это нормально. Подумай как сделать сравнение чуть мягче (подсказка: регистр и пробелы).

---

## Что должно получиться

1. `npm run generate-audio` — в `public/audio/` появляются `.ogg` файлы
2. Запускаешь повторно — файлы не перегенерируются
3. На карточке кнопка 🔊 — слышишь слово
4. Клик на 🔊 не переворачивает карточку
5. Кнопка 🎤 — говоришь перевод по-русски
6. Карточка переворачивается, видишь что сказала и попала ли
