# Домашнее задание 1: English Cards

## Проект

Будем делать платформу для изучения английских слов. Через 1.5 месяца это будет приложение где:

- карточки хранятся в базе данных и генерируются Python-скриптом через LLM
- зарегистрированные пользователи видят только те карточки, которые ещё не отметили "выучил"
- незарегистрированные — случайные карточки
- есть авторизация через NextAuth

Стек: **Next.js, PostgreSQL (Docker), Prisma, NextAuth, Python**.

В будущем (за рамками курса): статистика прогресса, вопросы с вариантами ответов, темы и категории, аудио.

---

## Домашка 1

**Цель:** новый репозиторий, база данных, три тестовые карточки, страница где можно их листать и переворачивать.

---

## Часть 1: Репозиторий

Создай новый репозиторий на GitHub — назови его `english-cards`. Инициализируй Next.js приложение:

```bash
npx create-next-app@latest .
```
Сразу убедись что в `.gitignore` есть строчка `.env` — она туда попадает автоматически, но проверь.

---

## Часть 2: База данных

В корне проекта создай `docker-compose.yml`, как мы уже делали.
Запусти:

```bash
docker compose up -d
```

В Docker Desktop контейнер `db` должен быть зелёным.
---

## Часть 3: Prisma

Мы уже работали с Prisma — повтори те же шаги для нового проекта:

```bash
npx prisma init
```
Открой `prisma/schema.prisma` и добавь модель карточки:

word, translation, transcription, example что-то еще как считаешь нужным

Примени схему и сгенерируй клиент:

```bash
npm run prisma:push
npm run prisma:generate
```

## Часть 4: Seed — тестовые данные
Т.е. мы хотим заполнить базу тестовыми карточками, чтобы было что показывать на странице.
Создай файл `prisma/seed.ts` с тремя карточками. Используй `prisma.card.createMany()` — документацию посмотри на сайте Prisma.
Карточки придумай сама — любые три английских слова которые хочешь выучить.

Запусти seed:

```bash
npx ts-node prisma/seed.ts
```

Проверь в Prisma Studio что три записи появились:

```bash
npm run prisma:studio
```

---

## Часть 5: Страница с карточками

Серверный экшн достаёт все карточки из базы. Страница вызывает экшн и передаёт карточки через props клиентскому компоненту `CardViewer`.

На странице `/` нужно показывать **одну карточку** с кнопками "←" и "→" для переключения между карточками.

Карточка переворачивается по клику: лицевая сторона — слово и транскрипция, обратная — перевод и пример предложения.

Подсказка: не забудь сбрасывать состояние флипа при переключении карточки — иначе следующая карточка будет сразу показана "перевёрнутой".

---

## Что должно получиться

1. `docker compose up -d` — база запускается
2. `npx ts-node prisma/seed.ts` — три карточки в базе
3. `npm run dev` — открываешь браузер
4. Видишь карточку со словом и транскрипцией
5. Кликаешь — она переворачивается, видишь перевод и пример
6. Нажимаешь "→" — следующая карточка (уже не перевёрнутая)
